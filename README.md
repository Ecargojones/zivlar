# ZIVLAR.COM - 环球别墅灵感平台 🌍🏡

这是一个基于 Next.js + Supabase 构建的全球乡村别墅灵感导航平台。

## 技术栈
- Next.js 14
- Tailwind CSS
- Supabase（Auth + Storage + Database）
- Cloudflare DNS + Vercel 托管
- next-i18next 多语言支持
- next-sitemap 自动生成 SEO 文件

## 快速启动

```bash
npm install
cp .env.example .env.local
npm run dev
```

## 部署说明

- 项目部署到 Vercel（推荐）
- 域名通过 Cloudflare 托管（zivlar.com）
- 自动生成 sitemap 和 robots.txt
